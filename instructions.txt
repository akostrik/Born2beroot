2) Debian
* The basic differences between Rocky and Debian
* Debian:
  multiple architecture support
  a vasy amount of packages in its default repository
  easily upgrated
  user-friendly applications and GUI (Graphical User Interface)
* apt = Advanced Packaging Tool = lower-level package manager, can be used by other higher-level package managers
  command line, no GUI 
  finds in /etc/apt/sources.list the package and its dependencies, automatically installs them
  easy control of configurations
* aptitude = high-level package manager
  a user interface to the functionality, interactively search for a package, install, remove it
  Can emulate most of apt-get’s command line arguments
  Abstracts low level details, can operate in both text-based interactive UI mode and even in command line non-interactive mode.
* AppArmor doit rester actif
  Associer a chaque programme un profil de securite qui restreint acces aux systeme d'exploitations
  Il est en complement du systeme de controle d'acces Linux en permettant d'utiliser le MAC : Mandatory Access control
  Permet un mode d'aprentissage
  Difference avec SELinux : travaille avec les chemins alors que SELinux s'appuie sur l'application d'indicateurs aux fichiers
  Программный инструмент упреждающей защиты, основанный на политиках (профилях) безопасности - к каким системным ресурсам и с какими привилегиями может получить доступ приложение
* both handle all kinds of activities on packages (installation, removal, search ...)
  https://www.tecmint.com/difference-between-apt-and-aptitude/

4) pam = pluggable authentification method
Strong password policy
$ sudo apt-get install libpam-pwquality	password quality checking library
/etc/pam.d/common-password:
  password requisite pam_pwquality.so retry=3 lcredit =-1 ucredit=-1 dcredit=-1 maxrepeat=3 usercheck=0 difok=7 enforce_for_root
   // dcredit=-1 credit=-1 credit=-1		an uppercase letter, a lowercase letter, and a number
   // maxrepeat=3												pas plus de 3 caracteres consecutifs
   // usercheck=1 											ne doit pas contenir le nom de l'utilisateur
   // difok=7														7 characters are not part of the former password 
   // enforce_for_root									apply to the root password
  password [success=2 default=ignore] pam_unix.so obscure sha512 minlen=10
  // default = not permit the user access to a service if the password is blank
/etc/login.defs 
		PASS_MAX_DAYS 30j										expire tous les 30j, 2j minimum pour modifier, un avertissement 7j
$ sudo reboot
$ chage -l akostrik											test
* the advantages of this password policy
* the advantages and disadvantages of its implementation

7) UFW = Uncomplicated Firewall
Le trafic entrant est refuse, le trafic sortant est autorise
$ apt update -y
$ apt install ufw -y
* /etc/default/ufw IPV6=yes
$ ufw enable
$ ufw default allow (deny)											autorise (refuse) le trafic entrant
$ ufw default allow (deny) ongoing							autorise (refuse) le trafic sortant
$ ufw allow 4242																ne laisser que le port 4242 ouvert
$ ufw deny 8080
$ ufw status numbered
$ ufw delete 4 																	delete rule 4
$ ufw allow ssh
